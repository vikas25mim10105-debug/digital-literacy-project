## Project Overview

The Digital Literacy Project is, about figuring out what digital literacy is about these days. This Digital Literacy Project has a lot of tasks that teach us about tools and how to actually use them. We learn about staying safe on the internet and how to work with people online. The main goal of the Digital Literacy Project is to help us become digital users so we can use the Digital Literacy Project to our advantage and become really effective digital users.
